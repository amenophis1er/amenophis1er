# Hey, I'm Amen

I build tools for voice, telecom, and the web.

## What I'm working on

**[x-phone](https://github.com/x-phone)** — Open-source telecom tooling for Go. SIP, VoIP, RTP — tested, scriptable, no infrastructure required.

**[oxiclean](https://github.com/getoxiclean/oxiclean)** — macOS storage cleaner. Find and remove hidden bloat — package caches, Docker artifacts, orphaned app data.

**[tts-bridge](https://github.com/amenophis1er/tts-bridge)** — TTS APIs speak HTTP. Voice apps speak WebSocket. This bridges the gap.

**[shopcart](https://github.com/amenophis1er/shopcart)** — Chrome extension to save Amazon products to custom lists.

## Stack
